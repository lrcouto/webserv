Como você chegou nesta página?

Quando você clicou no botão na página anterior, o seu browser enviou uma mensagem para o nosso servidor, formatada seguindo um padrão específico chamado de Hypertext Transfer Protocol - o tal do HTTP.

Através deste protocolo, o cliente (seu browser) envia uma mensagem ao servidor (nosso Webserv), pedindo para que seja entregue um recurso específico: esta página!

Agora, abra a janela de desenvolvedor do seu browser, apertando F12, e selecione a aba Rede, ou Network caso seu browser esteja em inglês. Em seguida, aperte o botão abaixo para prosseguir.
