Método DELETE

O terceiro método HTTP disponível no Webserv é o DELETE, usado para remover uma entidade do servidor. No entanto, você pode notar que no arquivo de configuração usado para subir o servidor, existe uma diretiva limit_except associada a este diretório, que permite que sejam usados apenas os métodos GET e POST dentro dele.

O que acontece se tentarmos deletar uma entidade contida no diretório methods? Clique no botão abaixo para enviar uma requisição DELETE para o endereço da imagem do upload da página anterior.
